git内网服务器搭建安装
==========================

使用的是 gitblit

1. java安装

2. 下载 gitblit http://www.gitblit.com/

3. 解压缩，不用安装 D:\installs\gitblit-1.8.0

4. 创建用于存储资料的文件夹 D:\gitRepository

5. 配置变量  gitblit.properties

6. 运行gitblit.cmd
